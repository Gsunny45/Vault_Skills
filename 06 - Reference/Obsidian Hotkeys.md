# Obsidian Hotkeys

## Essential Default Hotkeys

| Action | Windows/Linux |
|---|---|
| Open command palette | Ctrl+P |
| Quick switcher (open note) | Ctrl+O |
| Search in vault | Ctrl+Shift+F |
| New note | Ctrl+N |
| Save (auto-saves, but force) | Ctrl+S |
| Toggle edit/preview | Ctrl+E |
| Open settings | Ctrl+, |
| Close current tab | Ctrl+W |
| Navigate back | Alt+← |
| Navigate forward | Alt+→ |
| Toggle left sidebar | Ctrl+\ |
| Toggle right sidebar | Ctrl+/ |
| Open graph view | Ctrl+G |
| Toggle bold | Ctrl+B |
| Toggle italic | Ctrl+I |
| Insert internal link | Ctrl+K (then type) |
| Indent list item | Tab |
| Unindent list item | Shift+Tab |
| Fold/unfold heading | Ctrl+` |
| Split pane right | Ctrl+\ (customizable) |
| Toggle checklist | Ctrl+L |

## Tips
- All hotkeys are customizable in Settings → Hotkeys
- Use Command Palette (Ctrl+P) to find any action and assign a hotkey
- Plugins add their own hotkeys — check each plugin's settings
